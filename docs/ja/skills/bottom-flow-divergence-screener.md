# Bottom Flow Divergence Screener
{: .no_toc }

Screen the US market for "bottom + flow divergence" — stocks whose PRICE is on the floor (near the 52-week low, deep below the 52-week high) while a FLOW signal refuses to confirm that floor. Two layers: fundamental (TTM revenue still growing + positive operating cash flow — the business never broke, HOOD-type) and accumulation (Chaikin Money Flow / Money Flow Index positive — smart money buying the lows, the contrarian MRNA-type). Assigns A / B-accum / B-fund grades with turning/recovering/survivability tags and emits a JSON + Markdown watchlist. Use when the user wants beaten-down reversal candidates, bottom-fishing with positive divergence, "stocks at the bottom but cash flows recovering", or "акции на дне, но потоки не падали / восстанавливаются". No API key. Detection-only.
