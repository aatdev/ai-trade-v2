# Save Note
{: .no_toc }

Сохранить результат, заметку, отчёт или артефакт в личную базу знаний MyNotes ($MYNOTES_DIR из окружения или .envrc, по умолчанию ~/Documents/MyNotes), автоматически подобрав категорию и субкатегорию. Используй когда пользователь просит "сохрани это", "запиши в заметки", "сохрани результат", "положи в MyNotes" или хочет сохранить итог работы (анализ, исследование, конспект, инструкцию, код-сниппет) для последующего использования.
{: .fs-6 .fw-300 }
